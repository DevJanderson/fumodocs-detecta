![Logo Detecta](/logosdocs.svg)

## Como fazer em um documento MDX

Você pode adicionar imagens facilmente em seus documentos MDX. Por exemplo:

```md
![Logo Detecta](/logosdocs.svg)
```

Ou usando HTML para mais controle:

```html
<img src="/logosdocs.svg" alt="Logo Detecta" width="200" />
```
